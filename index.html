Myservy Design System

Un sistema de diseño de Myservy (Design Tokens), llamado Monochromatic Neutrals y tipografías/formas basadas en las guías oficiales de Material Design.

code Compilado con Style Dictionary v4
integration_instructions Compatible con CSS / SCSS
smartphone Diseño Figma Token Studio

¿Cómo funciona?

Los tokens se definen en formato JSON estandarizado en la carpeta /tokens. Usando Style Dictionary v4, procesamos estos tokens y generamos:

  • CSS Variables: Para cambios dinámicos de temas en el navegador.
  • SCSS Variables: Para compilación estática e integración óptima en proyectos Angular.
  • JS Variables: Para su uso dinámico en componentes TS/JS.

Instrucciones de Compilación

Para construir el Design System en tu entorno local:

# 1. Instalar dependencias con pnpm (evitando scripts bloqueados)
pnpm install --ignore-scripts

# 2. Generar tokens compilados
pnpm run build

Paleta de Colores (Monochromatic Neutrals)

Esta paleta utiliza tonos neutros y variaciones monocromáticas para crear un contraste, limpio y moderno. Los colores cambian dinámicamente según el tema seleccionado (Claro / Oscuro).

Uso de los colores

Primary Color var(--primary-color) Contenedores Primarios
Secondary Color var(--secondary-color) Botones seleccionados, tablas
Tertiary Color var(--tertiary-color) Textos principales, Iconos, Botones
Background var(--background) Fondos de pantalla
